Welcome to SMB

SMB is a financing webpage where you as a user or admin can interact with this website.
Down below you can read more about the different things you can do on this website.


User

As a user you can login with an existing user or create a new user. A user you can view, create and delete your banking accounts.


Admin

As a admin you have privilage to search for users, where you can see their password and if they have admin privilage or not.
You can edit users bank accounts as well.
